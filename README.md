 Project Description

Viitor is a web application built with **Next.js and React** that aggregates and displays the latest cryptocurrency news from multiple RSS feeds. The platform fetches articles from sources such as CoinDesk and CoinTelegraph and presents them in a clean, modern interface.

The project demonstrates front-end development with reusable React components, dynamic data fetching, and a structured application layout. It also includes basic pages for user authentication, navigation, and settings, showcasing a modular and scalable web application architecture.

 Technologies Used

- Next.js
- React
- JavaScript / TypeScript
- RSS feeds (CoinDesk, CoinTelegraph)
- CSS / Tailwind-style utility classes
